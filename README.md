# Arquitetura de Microsserviços

## Como rodar

Subir os serviços:
   ```bash
   docker-compose up -d
   ```

## Testar API

Execute os comandos de teste um por vez do arquivo `curls.sh`, copiando os IDs retornados para substituir nos próximos comandos.

## Limpar tudo

```bash
docker-compose down -v
```
